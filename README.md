# Loja Faculdade

Uma loja simples desenvolvida na uninove

# Como rodar?

Crie o seu banco de dados a partir dos scripts na pasta sql/

Execute os seguintes comandos:

```sh
cd "diretório do projeto"
php -S localhost:8080
```
